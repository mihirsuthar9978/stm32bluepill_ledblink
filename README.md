# STM32 LED Blink (Blue Pill)

## 📌 Overview

This project demonstrates basic GPIO control using STM32F103C8 (Blue Pill) with HAL drivers.

## ⚙️ Features

* GPIO output control
* LED blinking using HAL
* Delay using SysTick

## 🧠 Concepts Used

* Embedded C
* STM32 HAL
* GPIO configuration
* Microcontroller timing

## 🔌 Hardware

* STM32F103C8 (Blue Pill)
* Onboard LED (PA10)

## 🛠️ Tools Used

* STM32CubeIDE
* Proteus for simulation 

## ▶️ How It Works

The LED connected to PA10 is toggled every 500 ms using HAL functions.

## 🚀 Future Improvements

* Button input
* Interrupt-based control
* Timer-based blinking
